# Classification

Задача классификации: предсказание категории объекта и разделение объектов согласно определенным и заданным заранее признакам.

В работе успешно проведен полный цикл машинного обучения: от детального анализа данных и предобработки (кодирование, масштабирование) до обучения и тестирования трех моделей. Выполнен подбор оптимальных параметров с помощью GridSearchCV.
По результатам сравнения метрик, модель SVM является наиболее подходящей для данной задачи прогнозирования, демонстрируя наилучший баланс между точностью и способностью верно обнаруживать оба класса.
